Теория

CSS-селекторы позволяют выбирать элементы для стилизации. Основные типы: тег, класс, id, универсальный. Цвета задаются HEX, RGB, HSL или словами.

  • Тег: p — все абзацы
  • Класс: .main — все элементы с классом main
  • ID: #header — элемент с id="header"
  • Универсальный: * — все элементы

Лайфхак

Используй :hover для эффектов при наведении, :active — при нажатии.

p {
  color: #39ff14;
}
a {
  color: #8ee4af;
  text-decoration: underline;
  transition: color 0.2s;
}
a:hover {
  color: #21b573;
}

Теория

Свойства font-family, font-size, font-weight отвечают за шрифты. margin — внешний отступ, padding — внутренний. text-align выравнивает текст.

Лайфхак

Используй Google Fonts для стильных шрифтов. Например: @import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@700&display=swap');

body {
  font-family: 'Montserrat', Arial, sans-serif;
}
h1 {
  margin-bottom: 20px;
  padding: 10px;
  text-align: center;
  font-weight: 700;
}

Теория

Flexbox — современный способ выравнивания и распределения элементов. display: flex превращает контейнер в flex-контейнер.

  • justify-content — выравнивание по главной оси
  • align-items — по поперечной оси
  • gap — расстояние между элементами

Лайфхак

Flexbox отлично подходит для адаптивных меню и карточек.

.flex-demo {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 16px;
}
.flex-demo div {
  background: #39ff14;
  color: #232d25;
  padding: 12px 24px;
  border-radius: 10px;
}

Теория

Свойство transition делает изменения плавными. @keyframes позволяет создавать сложные анимации.

Лайфхак

Используй transform: scale() для эффекта "нажатия" на кнопку.

button {
  transition: background 0.3s, transform 0.2s;
}
button:hover {
  background: #39ff14;
  transform: scale(1.05);
}
@keyframes fadeIn {
  from { opacity: 0; }
  to   { opacity: 1; }
}

Теория

Медиа-запросы позволяют менять стили в зависимости от размера экрана. Используй @media для адаптива.

Лайфхак

Используй относительные единицы (em, rem, %) для гибкости.

@media (max-width: 600px) {
  body {
    font-size: 0.9rem;
    padding: 8px;
  }
  .main-btn {
    width: 100%;
    font-size: 1rem;
  }
}
Измени размер окна — текст станет меньше!

Теория

CSS-переменные (--var) позволяют легко менять цвета и стили во всём проекте. Используй их для темизации.

Лайфхак

Храни основные цвета и размеры в :root и меняй их для разных тем!

:root {
  --main-color: #39ff14;
  --bg: #232d25;
}
body {
  background: var(--bg);
  color: var(--main-color);
}
Этот текст и фон используют CSS-переменные!

Вопрос:

Что такое CSS-переменная?

Задание:

Вынеси основные цвета и размеры сайта в переменные :root и используй их в стилях.
